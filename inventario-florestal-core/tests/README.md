# Testes Automatizados

## Objetivo

Garantir estabilidade matemática e estatística do sistema biométrico.

## Tipos de teste

- regressão linear;
- regressão não linear;
- parser YAML;
- ranking;
- resíduos;
- taper;
- Weibull;
- prognose.

## Diretriz

Toda nova equação deve possuir:

- teste sintético;
- teste numérico;
- teste de estabilidade;
- validação dimensional.
