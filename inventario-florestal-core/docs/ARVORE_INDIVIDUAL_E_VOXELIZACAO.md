# Árvore Individual e Voxelização

## Objetivo

Documentar a arquitetura de árvore individual, voxelização e deep learning aplicada ao inventário remoto.

## Conceito

A biometria remota avançada deverá identificar árvores individualmente.

## Componentes

- segmentação individual;
- voxelização;
- deep learning;
- biomassa;
- carbono.

## Fluxo

LiDAR -> CHM -> segmentação -> voxelização -> árvore individual.

## Aplicações

- altura individual;
- copa;
- biomassa;
- carbono;
- competição.

## Conclusão

Árvore individual e voxelização representam a evolução para inventário remoto automatizado de alta precisão.
