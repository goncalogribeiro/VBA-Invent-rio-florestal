# Estratégia de Testes Científicos

## Objetivo

Documentar a estratégia de validação científica da plataforma.

## Tipos de Teste

### Testes Unitários

Validar:

- funções;
- regressões;
- cálculos biométricos.

### Testes de Integração

Validar:

- pipelines;
- persistência;
- integração espacial;
- integração temporal.

### Testes Ecológicos

Validar:

- crescimento plausível;
- mortalidade plausível;
- competição coerente;
- carbono plausível.

### Testes Ecofisiológicos

Validar:

- LAI;
- GPP;
- NPP;
- radiação;
- vigor.

### Testes Operacionais

Validar:

- desempenho;
- estabilidade;
- memória;
- reprocessamento.

## Datasets Prioritários

- inventário real;
- remedições reais;
- dados LiDAR;
- cenários de manejo;
- parcelas permanentes.

## Conclusão

Os testes científicos deverão validar a plataforma em condições operacionais reais.
