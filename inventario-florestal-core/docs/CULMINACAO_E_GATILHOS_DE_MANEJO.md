# Culminação e Gatilhos Automáticos de Manejo

## Objetivo

Documentar a arquitetura de culminação produtiva e gatilhos automáticos de manejo.

---

# Conceito

O manejo florestal moderno deve responder dinamicamente:

- à produtividade;
- à competição;
- à estabilidade;
- ao crescimento.

---

# Culminação do IMA

## Conceito

A culminação ocorre quando:

```text
ICA ≈ IMA
```

Representa:

- máxima eficiência volumétrica;
- referência clássica de rotação técnica.

---

# Interpretação

| Situação | Interpretação |
|---|---|
| ICA > IMA | crescimento acelerado |
| ICA ≈ IMA | culminação |
| ICA < IMA | desaceleração |

---

# Uso Silvicultural

A culminação poderá ser usada para:

- sugerir corte final;
- sugerir desbastes;
- avaliar prolongamento de rotação.

---

# Gatilhos Automáticos

O sistema deverá utilizar múltiplos gatilhos.

| Gatilho | Uso |
|---|---|
| Reineke | competição |
| Hart-Becking | ocupação |
| ICA | crescimento atual |
| IMA | produtividade média |
| sítio | potencial produtivo |
| estabilidade | risco estrutural |

---

# Regras Futuras

O sistema poderá recomendar:

| Situação | Recomendação |
|---|---|
| superlotação | desbaste |
| baixa ocupação | adiar manejo |
| ICA em queda | avaliar intervenção |
| culminação | avaliar corte |

---

# Integração com Prognose

```text
crescimento
↓
competição
↓
ICA/IMA
↓
culminação
↓
decisão silvicultural
```

---

# Observação Técnica

O sistema não deverá recomendar manejo apenas por idade fixa.

As decisões deverão considerar:

- estado estrutural;
- produtividade;
- competição;
- sítio.

---

# Evolução Futura

Futuramente o sistema poderá incorporar:

| Camada | Futuro |
|---|---|
| otimização econômica | ✓ |
| valor presente líquido | ✓ |
| custo operacional | ✓ |
| multiobjetivo | ✓ |
| carbono | ✓ |

---

# Conclusão

A culminação dinâmica e os gatilhos automáticos representam a camada responsável por transformar a prognose em suporte real à decisão silvicultural.
