# Testes Operacionais de Manejo

## Objetivo

Documentar os testes operacionais mínimos de manejo/desbaste necessários para consolidar a plataforma.

## Diretriz de Escopo

Não criar novo motor de manejo avançado nesta fase.

Implementar apenas o necessário para testar estabilidade pós-intervenção.

## Intervenção Inicial

A primeira intervenção operacional será o desbaste seletivo simplificado por número de árvores.

## Critérios de Remoção

A remoção deverá priorizar árvores estruturalmente inferiores:

1. suprimidas;
2. dominadas;
3. codominantes;
4. dominantes somente se necessário.

## Validações

Após o desbaste, validar:

- redução da população;
- ausência de valores negativos;
- estabilidade pós-intervenção;
- continuidade da simulação temporal;
- sobrevivência remanescente.

## Conclusão

Os testes operacionais de manejo têm a função de verificar se a prognose permanece estável após perturbações estruturais controladas.
