# GPP, NPP e Respiração

## Objetivo

Documentar a arquitetura de produtividade primária, respiração e balanço de carbono.

## Componentes

- GPP;
- NPP;
- respiração autotrófica;
- balanço de carbono;
- eficiência energética.

## Fluxo

radiação -> fotossíntese -> GPP -> respiração -> NPP.

## Aplicações

- produtividade;
- carbono;
- ecofisiologia;
- prognose.

## Conclusão

GPP, NPP e respiração representam a evolução para modelagem fisiológica funcional completa.
