# Stress Tests Severos

## Objetivo

Registrar os stress tests severos aplicados ao simulador temporal.

## Cenários Avaliados

| Cenário | Horizonte |
|---|---|
| normal | 25 anos |
| elite genética | 25 anos |
| sítio ruim | 25 anos |
| superlotado | 10 anos |

## Verificações Executadas

- DAP positivo;
- altura positiva;
- idade positiva;
- ausência de explosão estrutural;
- ausência de colapso total;
- sobrevivência plausível.

## Limites Temporários de Auditoria

| Variável | Limite |
|---|---|
| DAP | < 150 cm |
| Altura | < 80 m |

## Objetivo Científico

Verificar se a integração entre:

- crescimento;
- competição;
- dominância;
- mortalidade;
- perfis silviculturais;

permanece estável em horizontes temporais longos.

## Conclusão

Os stress tests severos representam a primeira validação longitudinal profunda da plataforma.
