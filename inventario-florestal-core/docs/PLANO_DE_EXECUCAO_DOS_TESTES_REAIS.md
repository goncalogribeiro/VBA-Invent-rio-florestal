# Plano de Execução dos Testes Reais

## Objetivo

Definir o fluxo operacional da validação científica da plataforma.

## Fase 1 — Baseline

Objetivo:

- validar inventário;
- validar biometria;
- validar persistência.

### Entradas

- inventário real;
- parcelas reais;
- DAP;
- altura;
- idade;
- manejo.

### Saídas

- volume;
- área basal;
- altura estimada;
- consistência biométrica.

## Fase 2 — Prognose

Objetivo:

- validar crescimento;
- validar mortalidade;
- validar competição.

### Entradas

- remedições;
- crescimento observado;
- mortalidade observada.

### Saídas

- crescimento projetado;
- mortalidade projetada;
- estabilidade temporal.

## Fase 3 — Manejo

Objetivo:

- validar regimes;
- validar desbaste;
- validar sortimento.

### Cenários

- sem desbaste;
- leve;
- moderado;
- pesado;
- misto.

## Fase 4 — Ecofisiologia

Objetivo:

- validar LAI;
- validar GPP;
- validar NPP;
- validar carbono.

## Fase 5 — Aprendizado Adaptativo

Objetivo:

- validar recalibração;
- validar estabilidade;
- validar ausência de drift.

## Critérios de Aprovação

- estabilidade matemática;
- plausibilidade biológica;
- coerência ecológica;
- estabilidade temporal;
- rastreabilidade.

## Conclusão

O plano deverá permitir validação progressiva e auditável da plataforma.
