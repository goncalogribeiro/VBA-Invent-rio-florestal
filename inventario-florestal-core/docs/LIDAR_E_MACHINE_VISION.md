# LiDAR e Machine Vision

## Objetivo

Documentar a arquitetura LiDAR, raster climático e machine vision integrada à plataforma.

## Conceito

A biometria florestal moderna combina inventário de campo e sensoriamento remoto.

## Componentes

- LiDAR;
- nuvem de pontos;
- raster climático;
- machine vision;
- segmentação espacial.

## Aplicações

- altura florestal;
- estrutura vertical;
- carbono;
- mortalidade;
- vigor;
- risco.

## Integração

campo -> LiDAR -> analytics -> prognose.

## Conclusão

LiDAR e machine vision representam a evolução da plataforma para biometria remota inteligente.
