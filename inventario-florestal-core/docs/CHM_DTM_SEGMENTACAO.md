# CHM, DTM/DSM e Segmentação Individual

## Objetivo

Documentar a arquitetura de modelos de altura, terreno e segmentação individual de árvores.

## Componentes

- CHM;
- DTM;
- DSM;
- segmentação de copas;
- biomassa;
- carbono.

## Fluxo

LiDAR -> DTM/DSM -> CHM -> segmentação -> biometria.

## Aplicações

- altura individual;
- copa;
- biomassa;
- carbono;
- mortalidade.

## Conclusão

CHM e segmentação representam a evolução para inventário remoto automatizado.
