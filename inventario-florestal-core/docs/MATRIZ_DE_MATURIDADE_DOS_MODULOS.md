# Matriz de Maturidade dos Módulos

## Objetivo

Classificar o nível de maturidade real dos módulos da plataforma.

## Critérios

| Status | Definição |
|---|---|
| completo | funcional e integrado |
| parcial | funcional simplificado |
| mockado | estrutural sem modelagem completa |
| planejado | ainda não implementado |
