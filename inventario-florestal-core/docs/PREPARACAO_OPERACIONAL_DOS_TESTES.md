# Preparação Operacional dos Testes

## Objetivo

Documentar a preparação operacional da plataforma para validação científica.

## Estrutura dos Testes

### Dataset de Inventário

Deverá conter:

- parcelas;
- árvores;
- DAP;
- altura;
- idade;
- espécie;
- qualidade;
- posição espacial.

### Dataset Temporal

Deverá conter:

- remedições;
- crescimento observado;
- mortalidade observada;
- histórico de manejo.

### Dataset Espacial

Deverá conter:

- shapefiles;
- GeoJSON;
- LiDAR;
- CHM;
- raster.

### Dataset Ecofisiológico

Deverá conter:

- clima;
- radiação;
- LAI;
- NDVI;
- carbono.

## Cenários Obrigatórios

- sem desbaste;
- desbaste leve;
- desbaste moderado;
- desbaste pesado;
- manejo misto.

## Verificações Obrigatórias

- estabilidade;
- coerência;
- rastreabilidade;
- desempenho;
- plausibilidade.

## Conclusão

A preparação operacional deverá garantir testes científicos reproduzíveis e auditáveis.
