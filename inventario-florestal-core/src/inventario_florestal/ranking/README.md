# Motor Universal de Ranking

## Objetivo

Executar torneio estatístico universal entre modelos biométricos.

## Responsabilidades

- ajuste de modelos;
- cálculo de métricas;
- análise residual;
- correção de viés;
- ranking;
- seleção automática.

## Entradas

- dataframe de dados;
- definição YAML do modelo;
- configuração do torneio.

## Saídas

- coeficientes;
- métricas;
- resíduos;
- ranking final;
- modelo vencedor.
